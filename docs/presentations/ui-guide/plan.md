# UI操作ガイド・プレゼンテーション作成計画

## 概要

Novel-Athanor-v2 のUI（コマンド）操作方法を説明する視覚的なHTMLプレゼンテーションを作成する。

## 対象読者

- パソコンスキルあり
- Claude Code の設定は可能
- プロジェクト遂行は難しい（開発者ではない）
- 小説執筆可能（創作者）

→ **技術者ではない創作者向け** に分かりやすく説明する

## 言語方針

- 説明テキスト: 日本語
- コード例・コマンド名: 英語

---

---

## 設計方針

### 操作手順の可視化

- コマンド入力例をコードブロックで表示
- 出力例も併記
- スクリーンショット風のUIモックアップ

### 創作者向け表現

- 技術用語を最小限に
- 「何をしたいとき」→「このコマンド」の対応表
- 実際の執筆シーンに沿った例示
