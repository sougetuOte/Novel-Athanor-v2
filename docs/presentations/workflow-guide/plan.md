# 執筆ワークフロー・プレゼンテーション作成計画

## 概要

Novel-Athanor-v2 の執筆ワークフローを説明する視覚的なHTMLプレゼンテーションを作成する。

## 対象読者

- パソコンスキルあり
- Claude Code の設定は可能
- プロジェクト遂行は難しい（開発者ではない）
- 小説執筆可能（創作者）

→ **技術者ではない創作者向け** に分かりやすく説明する

## 言語方針

- 説明テキスト: 日本語
- コード例・モジュール名: 英語

## 設計方針

### ビジュアル重視

- フローチャートを多用（CSS/SVGで実装）
- エージェントアイコンで視覚的に役割を表現
- アニメーションは控えめ（読みやすさ優先）

### 創作者向け表現

- 技術用語を避け、創作者視点で説明
- 「なぜそうするのか」を明記
- 実際の執筆シーンに沿った例示
